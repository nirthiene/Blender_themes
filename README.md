## How to Install

1. Download the `.xml` file of your chosen theme from this repository.
2. Open Blender and go to **Edit -> Preferences -> Themes**.
3. Click the **Install...** button at the top right of the Preferences window.
4. Select the downloaded `.xml` file and click **Install Theme**.
